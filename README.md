## 🧩 Review Focus Areas

My Japanese Localization QA process is based on six core evaluation axes:

1. **Terminology** — Consistent and domain-appropriate terms  
2. **Naturalness** — Native-level fluency and readability  
3. **Consistency** — Uniform style across UI components  
4. **Brevity** — Concise, efficient Japanese phrasing  
5. **Context** — Correct meaning based on UI behavior  
6. **Tone** — Appropriate voice for SaaS products  

These principles ensure that Japanese UI text is **clear, natural, and aligned with product intent**.
